# M3U Editor Documentation

Official documentation for [M3U Editor](https://github.com/sparkison/m3u-editor) - A full-featured IPTV playlist manager.

Built with [Docusaurus](https://docusaurus.io/).

## ğŸš€ Quick Start

### Installation

```bash
npm install
```

### Local Development

```bash
npm start
```

This starts a local development server and opens a browser window. Most changes are reflected live without restarting.

### Build

```bash
npm run build
```

Generates static content into the `build` directory for deployment.

## ğŸ¨ Customization

### Custom CSS

Edit `src/css/custom.css` for custom styles:

```css
:root {
  --ifm-color-primary: #your-color;
}
```

### Custom Components

Create React components in `src/components/`:

```jsx
// src/components/MyComponent.jsx
export default function MyComponent() {
  return <div>Custom component</div>;
}
```

Use in markdown:

```markdown
import MyComponent from '@site/src/components/MyComponent';

<MyComponent />
```

## ğŸ”§ Troubleshooting

### Build Errors

Clear cache and rebuild:
```bash
npm run clear
npm run build
```

### Port Already in Use

Change the port:
```bash
npm start -- --port 3001
```

### Broken Links

Check for broken links:
```bash
npm run build
```

Docusaurus will fail the build if there are broken links (configured with `onBrokenLinks: 'throw'`).
